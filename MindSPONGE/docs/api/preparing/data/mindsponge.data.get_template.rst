mindsponge.data.get_template
============================

.. py:function:: mindsponge.data.get_template(template, residue_name=None)

    获取分子模板。

    参数：
        - **template** (Union[str, dict, list]) - 模板的文件名。
        - **residue_name** (str) - 残基名称。默认值："None"。

    输出：
        dict。分子模板。
