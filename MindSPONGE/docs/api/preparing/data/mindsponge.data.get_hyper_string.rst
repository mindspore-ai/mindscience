mindsponge.data.get_hyper_string
================================

.. py:function:: mindsponge.data.get_hyper_string(hyper_param, prefix)

    获取字符串类型的超参数。

    参数：
        - **hyper_param** (dict) - 超参数字典。
        - **prefix** (str) - 只有开头带有prefix的参数才会被加载。

    输出：
        str。字符串类型的超参数。
