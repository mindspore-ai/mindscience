mindsponge.data.get_bonded_types
================================

.. py:function:: mindsponge.data.get_bonded_types(atom_types, symbol="-")

    得到键、角、二面体等键项的类型。

    参数：
        - **atom_types** (ndarray) - 原子种类。
        - **symbol** (str) - 一个符号。默认值："-"。

    输出：
        ndarray。键、角、二面体等键项的类型。
