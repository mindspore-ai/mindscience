sponge.function.set_global_units
====================================

.. py:function:: sponge.function.set_global_units(length_unit, energy_unit, units)

    给全局单位设置单位。

    参数：
        - **length_unit** (str) - 长度单位。
        - **energy_unit** (str) - 能量单位。
        - **units** (str) - 单位。
