mindearth.utils.plt_metrics
==============================================

.. py:function:: mindearth.utils.plt_metrics(x, y, title, label, loc="upper right")

    根据给定的日期和数据索引获取数据文件名称。

    参数：
        - **x** (numpy.array) - 绘图中的横坐标。
        - **y** (numpy.array) - 绘图中的纵坐标。
        - **title** (str) - 绘图的标题。
        - **label** (str) - 绘图的标签。
        - **loc** (str) - 图例的位置。
