mindflow.core.batched_jacobian
===================================

.. py:function:: mindflow.core.batched_jacobian(model)

    计算网络模型的雅可比矩阵。

    参数：
        - **model** (mindspore.nn.Cell) - 输入维度为in_channels输出维度为out_channels的网络模型。

    返回：
        Hessian(Tensor)，用于计算雅可比矩阵的Jacobian实例。输入维度为：[batch_size，in_channels]，输出维度为：[out_channels，batch_size，in_channels]。

    .. note::
        要求MindSpore版本 >= 2.0.0调用如下接口： `mindspore.jacrev` 。
