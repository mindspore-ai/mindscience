# 二维Taylor Green涡流

## 概述

在流体力学中，Taylor-Green涡流动是一种不稳定的衰减的涡流，在二维周期性边界条件时存在精确解，该精确解与Navier-Stokes方程的解一致。本案例使用PINNs对2维的taylor green涡流进行仿真
